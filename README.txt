Binary Run
----------

An educational game for Computer Science



Directory Structure:

Binary Run.exe

- Pre-compiled executable that runs the game



Binary Run/

- Houses all of the various Godot project files

Binary Run/assets

- Contains all of the games various music, sprites and models

Binary Run/modules

- Contains the scenes and scripts for the two modules that appear in game


Binary Run/scenes

- Contains all of the scenes used within the project





How to Modify:

1. Launch the Godot Engine

2. Select import

3. Navigate to the "Binary Run" folder

4. Open the project.godot file
